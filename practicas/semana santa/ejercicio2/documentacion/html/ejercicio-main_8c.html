Documentación de las funciones

main ( )

Función principal.

El programa pedirá las dimensiones de la matriz que generará, también pedirá un número de corte para extraer los elementos mayores.
Las acciones se realizarán del siguiente modo:

  • Pedir el numero de filas y columnas
  • Reservar memoria para la matriz
  • Rellenar la matriz
  • Mostrar la matriz resultante
  • Pedir al usuario el elemento de corte a la hora de buscar en la matriz
  • Extraer el vector
  • Mostrarlo por pantalla
  • Calcular el máximo y mínimo
  • Imprimir dichos resultados
Devuelve
  • 0 si todo fue correcto
  • Otro valor si hubo algún fallo
